A simple database writing in rust, inspired from mit 6.830

## Roadmap

### 0.0.0

- B+ tree storage structure
- Buffer pool
- Insert
- No transaction
- No multi-threading
- No RAL (write ahead log)

### 0.1.0

- Publish to crates.io

### 0.2.0

- Iterator

### 0.3.0

- Delete, query, update

### 1.0.0

- SQL (used for TPC test)
- Page topology
